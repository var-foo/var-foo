Create a To Do list on the WordPress Dashboard.

== Description ==

Create a To Do list on the WordPress Dashboard. This tool is very interesting if you're writing posts with several authors.

The To Do list is in the theme of the WordPress admin pages so it will fit nice into your Dashboard!

Read more about the plugin, suggest features or do you have other comments, please visit the <a href="http://crispijnverkade.nl/blog/wordpress-easy-todo">plugin page</a>

Enjoy!

== Installation ==

1. Upload the folder wp-easy-todo to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Drag the to do list to the preferred position on your dashboard

== Upgrade ==

1. Delete the old plugin folder `wp-create-todo`.
1. Upload the folder wp-easy-todo to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Drag the to do list to the preferred position on your dashboard
